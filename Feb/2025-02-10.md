## 🐳 Docker 기본 명령어 정리

### 도커 버전 확인
 
docker version

### 도커 시스템 정보 확인
 
docker info

### 실행 중인 컨테이너 목록 확인
 
docker ps

 ### 모든 컨테이너(정지된 컨테이너 포함) 확인
 
docker ps -a

 ### 로컬에 있는 이미지 목록 확인
 
docker images

 ### Docker Hub에서 이미지 검색
 
docker search nginx

 ### Docker Hub에서 이미지 다운로드
 
docker pull ubuntu

 ### 로컬에서 이미지 삭제
 
docker rmi nginx

 ### 컨테이너 삭제
 
docker rm mycontainer
